Commands

EEWA

DataCmd ShowSettingsEEWA - Will bring up Admin Settings for EEWA
DataCmd EEWA_GivePlayerItem[PlayerName] [ItemID] [Quantity] - Will give the specified player the specified item of that ID and in that quantity.
DataCmd SpawnMeteors_EEWA [PlayerName] - Will spawn Oblivion Meteors nearby specified player. 
DataCmd DeleteAll_YmirSpawners - If you have trouble deleting a placed Ymir Spawner, this can be used to delete it and any others. 
DataCmd Activate_Surges_EEWA - If you have enabled Custom Surges, this will try and activate an EEWA Surge. 
RemoveSigil_EEWA [PlayerName] - Will remove specified players Sigils.
